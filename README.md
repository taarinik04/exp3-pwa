# 🚀 React PWA Application (Experiment 1.3)

A modern **Progressive Web App (PWA)** built using **React + Vite**, enhanced with **Material UI (MUI)** and configured for offline support using **vite-plugin-pwa**.

---

## 📌 Features

* ⚡ Fast development with **Vite**
* ⚛️ Built using **React 19**
* 🎨 UI powered by **Material UI (MUI)**
* 📱 Progressive Web App (PWA) support
* 🔄 Auto-update service worker
* 📦 Optimized production build
* 🧹 ESLint for clean and consistent code

---

## 🛠️ Tech Stack

* React
* Vite
* Material UI (MUI)
* Vite Plugin PWA
* ESLint

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/react-pwa-app.git
cd react-pwa-app
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Run development server

```bash
npm run dev
```

👉 Starts the app on `http://localhost:5173`

---

## 🚀 Build for Production

```bash
npm run build
```

---

## 👀 Preview Production Build

```bash
npm run preview
```

## 📱 PWA Configuration

This project is configured as a Progressive Web App using `vite-plugin-pwa` 

* Auto update enabled (`registerType: autoUpdate`)
* App manifest includes:

  * App name: **React PWA**
  * Theme color: `#ffffff`
  * Icons for installable app
